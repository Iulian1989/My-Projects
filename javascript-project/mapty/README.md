# 🗺️ Mapty – Workout Tracker

## 📌 About the project

This project was built as part of a JavaScript course. The goal was to practice working with real-world browser features like maps, geolocation, and local storage.

Even though it follows the course structure, I worked through the logic myself and understood how each part works.

## 🚀 What the app does

Mapty is a simple app where you can:

* track running and cycling workouts
* see them displayed on a map
* store workouts so they remain after page refresh

## 🧪 What I used

* JavaScript (classes & OOP concepts)
* HTML & CSS
* Leaflet.js (for the map)
* Geolocation API
* Local Storage

## ▶️ How to run

Just open `index.html` in your browser.

## 📁 Structure

* `script.js` → main logic
* `index.html` → layout
* `style.css` → styling

## 📚 What I learned

* how to structure an app using classes
* working with APIs (geolocation, maps)
* handling user input and updating the UI
* saving data in the browser

## 👨‍💻 Note

This project is based on a JavaScript course, but I used it to understand how a real application works and to practice writing cleaner code.
