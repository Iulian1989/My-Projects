# 🎲 Pig Game

## 📌 About the project

This project was built as part of a JavaScript course. It helped me understand how to work with DOM manipulation, game logic, and event handling in a practical way.

Even though it follows the course, I went through the logic step by step and made sure I understood how everything works.

## 🎮 What the app does

Pig Game is a simple dice game where:

* two players take turns rolling a dice
* the current score adds up with each roll
* if you roll a 1, you lose your current score
* you can choose to "hold" and save your score
* the first player to reach the target score wins

## 🧪 What I used
