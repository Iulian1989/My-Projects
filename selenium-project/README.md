# 🤖 Selenium UI Automation – SauceDemo

## 📌 About the project

This is a personal UI automation project built using Selenium. I used it to practice writing automated tests and understanding how Selenium works in real scenarios.

The tests are based on a demo website and focus on basic UI automation flows.

---

## 🌐 Website tested

https://www.saucedemo.com/

---

## 🚀 What is covered

* Opening the browser and navigating to the website
* Logging into the application
* Interacting with UI elements (buttons, inputs)
* Adding products to the cart
* Basic validations

---

## 🧪 Technologies used

* Selenium WebDriver
* Java
* Test automation basics

---

## ▶️ How to run

* Open the project in your IDE (e.g., IntelliJ)
* Run the test class: `SauceDemoTest.java.`

---

## 📁 Project structure

* `SauceDemoTest.java` → main test logic

---

## 📚 What I learned

* how Selenium interacts with web elements
* locating elements using selectors
* handling user actions (click, input)
* understanding test flow in UI automation

---

## 👨‍💻 Note

This is a learning project using a demo website. The focus was on understanding Selenium basics and how UI automation works.
