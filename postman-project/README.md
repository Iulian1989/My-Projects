# 🔌 API Testing with Postman

## 📌 About the project

This project contains API tests created using Postman. I used it to understand how API testing works in practice and how to validate responses from different endpoints.

The project is based on testing a demo API, focusing on learning rather than building a production-ready test suite.

---

## 🚀 What is covered

* Sending GET and POST requests
* Validating response status codes
* Checking response body and structure
* Using basic test scripts in Postman
* Organizing requests into collections

---

## 🧪 Tools used

* Postman
* JavaScript (for test scripts)

---

## ▶️ How to use

1. Import the Postman collection
2. Run requests individually or using Collection Runner

---

## 📁 Project structure

* Postman Collection → contains all API requests
* Environment (optional) → for variables

---

## 📚 What I learned

* how APIs work (requests & responses)
* validating status codes and data
* writing simple test scripts in Postman
* organizing API tests into collections

---

## 👨‍💻 Note

This is a learning project using a demo API. Some endpoints may have limitations, but the goal was to understand API testing fundamentals.
